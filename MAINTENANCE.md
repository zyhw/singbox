# sing.sh 维护文档

## 更新日志 (2026-02-04)

### `SOCKS_USER` 随机化

**变更内容**：
将固定的 SOCKS5 用户名 `admin` 修改为随机生成的 8 位十六进制字符串。

**变更位置**：
`sing.sh` 第 60 行左右。

**代码变更**：
```bash
- SOCKS_USER="admin"
+ SOCKS_USER=$(openssl rand -hex 4)
```

**目的**：
提高安全性，防止使用默认凭据。

### 维护指南

- **用户凭据**：脚本执行后，随机生成的用户名会包含在 `~/sing-box.txt` 的输出中。
- **验证**：可以通过运行脚本并检查输出或 `config.json` 中的 `users` 字段来验证生成的用户名是否为随机。
